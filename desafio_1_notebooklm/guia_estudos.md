**1º prompt:** Apesar de ser um curso de considerado para iniciantes, é necessário alguns requisitos básicos para entender profundamente o assunto. Sendo assim, a pergunta abaixo foi enviada:

*Sendo iniciante em programação (sem nenhuma experiência), por onde deveria começar a estudar? O que de fato é TypeScript e Node.js? Quais os requisitos prévio para estudar sobre esse assunto?*

A seguinte resposta foi obtida:

*Para quem está começando do zero na programação, o caminho ideal é iniciar pelos fundamentos básicos antes de avançar para ferramentas específicas. As fontes sugerem que tutoriais de nível intermediário, como a criação de CRUDs (sistemas de cadastro) ou autenticação, exigem que o estudante já tenha familiaridade prévia com a lógica e a linguagem base.
Aqui estão as explicações sobre as tecnologias e como você deve estruturar seus estudos:
O que são Node.js e TypeScript?
Node.js: Não é uma linguagem de programação, mas sim um ambiente de execução (runtime) que permite rodar o código JavaScript fora do navegador, ou seja, no seu próprio computador ou em servidores
. Ele é conhecido por ser eficiente, orientado a eventos e é composto por vários componentes internos, como o motor V8 (que interpreta o código) e a biblioteca libuv (que lida com tarefas assíncronas e acesso ao sistema operacional)
.
TypeScript: É um superset (um "superconjunto") de JavaScript que adiciona tipagem estática à linguagem
. Na prática, ele permite que você defina se uma informação é um texto, um número ou um objeto, ajudando a prevenir erros ainda durante a escrita do código e facilitando a manutenção de projetos grandes
.
Por onde começar a estudar?
Fundamentos de JavaScript: Como o Node.js executa JavaScript e o TypeScript é baseado nele, aprender a base da linguagem JavaScript é o primeiro passo essencial
.
Lógica e Web Básica: Entender como a internet funciona e como os navegadores se comunicam com servidores é fundamental para quem quer trabalhar com backend
.
Ambiente de Desenvolvimento: Comece instalando o Node.js (preferencialmente a versão LTS, que é mais estável) e um editor de código, como o Visual Studio Code (VS Code)
.
Requisitos Prévios
Para mergulhar nos assuntos abordados nas fontes, você precisará de:
Instalação do Node.js: Necessária para rodar qualquer aplicação JavaScript ou TypeScript fora do browser
.
Conhecimento básico de Terminal: Você precisará utilizar linhas de comando para instalar pacotes (usando o npm ou yarn) e inicializar seus projetos
.
Familiaridade com JavaScript e Express.js: Muitos conteúdos de backend utilizam o framework Express para gerenciar rotas e requisições HTTP, sendo um requisito comum para entender arquiteturas mais complexas
.
Inglês Técnico: Embora existam ótimos materiais em português, grande parte da documentação oficial e dos artigos técnicos avançados estão em inglês, sendo importante desenvolver a habilidade de leitura nesse idioma
.
Se você for totalmente iniciante, a recomendação é não começar diretamente por arquiteturas complexas, mas sim por tutoriais introdutórios de JavaScript e Node.js para construir uma base sólida*
