# ▶️ Caderno Temático utilizando Inteligência Artificial
> Desafio 01: Criação de um guia de estudos utilizando a ferramenta NotebookLM.

---

## 🎯 Objetivo
Criar um caderno de anotações sobre o desenvolvimento Back-end com foco em Node.js e TypeScript para auxiliar nos estudos durante o curso.
O objetivo é criar um auxiliador para conceitos introdutórios e fundamentais para programadores iniciantes.
